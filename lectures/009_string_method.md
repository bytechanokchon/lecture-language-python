### String function

- **len()** ใช้นับว่าใน string มีตัวอักษรกี่ตัว
- **.find(a)** ค้นหาตัวอักษรที่ระบุ จากข้างหน้าไปข้างหลัง และส่งคืนตำแหน่งแรกที่เจอตัวอักษร หากไม่เจอจะส่งคืน -1
- **.rfind(a)** ค้นหาตัวอักษรที่ระบุ จากข้างหลังไปข้างหน้า และส่งคืนตำแหน่งแรกที่เจอตัวอักษร หากไม่เจอจะส่งคืน -1
- **.capitalize()** ทำให้ตัวอักษรตัวแรกของประโยคเป็นตัวพิมพ์ใหญ่ ที่เหลือเป็นตัวพิมพ์เล็ก
- **.upper()** ทำให้ตัวอักษรเป็นตัวใหญ่ทั้งหมด
- **.lower()** ทำให้ตัวอักษรเป็นตัวเล็กทั้งหมด
- **.isdigit()** ตรวจว่าเป็นตัวเลขทั้งหมดหรือไม่
- **.isalpha()** ตรวจว่าเป็นอักษรทั้งหมดหรือไม่
- **.count(v)** นับว่าตัวอักษรที่ระบุมีกี่ตัวใน string
- **.replace(a, b)** เขียนทับเป็นอักษรที่ระบุ (a) ด้วยตัวอักษรที่เรากำหนด (b) และส่งคืน string ใหม่

*ฟังก์ชันปกติที่ขึ้นต้นด้วย r จะหมายถึงฟังก์ชันนั้นทำงานย้อนกลับจากที่มันควรจะเป็น โดย r ย่อมาจาก reverse*

*ใช้คำสั่ง print(help(str)) เพื่อดูฟังก์ชันทั้งหมดที่สามารถใช้ได้กับ string*
