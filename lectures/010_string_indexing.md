### string indexing
เป็นการที่เราสามารถเข้าถึงข้อมูลใน string ด้วยการระบุตำแหน่งที่กำหนดโดยใช้สัญลักษณ์ [] (indexing operator) โดยสามารถระบุได้ดังนี้

    [ start : stop : step ]

โดยแต่ละช่วงคือ
- **start** ตำแหน่งเริ่มต้นของการเข้าถึงข้อมูล
- **stop** ตำแหน่งสิ้นสุดของการเข้าถึงข้อมูล (ข้อมูลตัวสุดท้ายจะไม่ถูกเข้าถึง)
- **step**

*ตำแหน่งแรกจะเริ่มต้นที่ 0*
