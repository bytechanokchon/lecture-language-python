### If statement
ใช้สำหรับควบคุมเงื่อนไขการทำงานของชุดคำสั่ง โดยมีรูปแบบดังนี้

    if  เงื่อนไข:
        สิ่งที่จะทำหากเงื่อนไขเป็นจริง
    elif เงื่อนไข:
        สิ่งที่จะทำหากเงื่อนไขเป็นจริง
    else:
        สิ่งที่จะทำหากเงื่อนไขเป็นเท็จ

elif (else if) จะใช้ในกรณีที่หากเงื่อนไขที่ต้องการตรวจสอบมีหลายเงื่อนไข
