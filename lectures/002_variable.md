### Variable
ตัวแปร เป็นสิ่งที่เอาไว้จัดเก็บข้อมูล โดยรูปแบบของข้อมูลที่จัดเก็บได้จะมีดังนี้
1. **string** จัดเก็บข้อมูลในรูปแบบข้อความ
2. **integer** จัดเก็บข้อมูลในรูปแบบเลขจำนวนเต็ม
3. **float** จัดเก็บข้อมูลในรูปแบบเลขจำนวนเต็มพร้อมทศนิยม
4. **boolean** จัดเก็บข้อมูลจริง/เท็จ

โดยการตั้งชื่อให้กับตัวแปร จะต้องไม่ซ้ำกัน

โดยปกติในภาษา python จะเป็นแบบ dynamic type ซึ่งชนิดข้อมูลของตัวแปรจะขึ้นอยู่กับข้อมูลที่มันเก็บ
แต่เราสามารถระบุชนิดของข้อมูลที่ต้องการจะเก็บไว้ในตัวแปรนั้นจริง ๆ ได้ โดยใช้รูปแบบดังนี้

    variable_name: type = value

โดยชนิดข้อมูลมีดังนี้
1. int
2. str
3. float
4. bool
