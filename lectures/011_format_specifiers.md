### Format Specifiers
ตัวกำหนดรูปแบบการแสดงผลข้อมูล โดยควบคุมว่าข้อมูลที่จะถูกแปลงเป็นข้อความ จะออกมาอย่างไร เช่น จำนวนจุดทศนิยม ความกว้าง การจัดชิดซ้าย-ขวา หรือ รูปแบบตัวเลข

รูปแบบ

    {value:flags}

ประเภทของ flags
- **.(number)f** ระบุตำแหน่งทศนิยมที่ต้องการแสดง
- **:(number)** กำหนดขนาดช่องสำหรับการแสดงผล โดยนับรวมกับข้อมูลด้วย
- **:0(number)** ใช้ร่วมกับการดพหนดขนาดช่องสำหรับแสดงผล โดยพื้นที่ว่างที่เหลือจะทำการเติมเลข 0 เข้าไป 
- **:<** ชิดซ้าย
- **:>** ชิดขวา
- **:^** กลึ่งกลาง
- **:+** ตัวเลขที่มากกว่า 0 จะเติมเครื่องหมาย + ไว้ด้านหน้า
- **:,** ให้มีการขั้นตัวเลขด้วยคอมม่า (,) ตามหลักคณิตศาสตร์
